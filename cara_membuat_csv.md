Ini adalah panduan lengkap untuk membuat file .csv (Comma Separated Values) yang berisi soal-soal kuis Anda.Aturan Format (Wajib Diikuti)Baris pertama (header) harus persis seperti ini:q,o,a,difficultyq: Teks pertanyaan Anda.o: Teks opsi jawaban. HARUS dipisahkan dengan \\ (backslash).a: Teks jawaban yang benar. HARUS sama persis dengan salah satu opsi.difficulty: Angka tingkat kesulitan. 1 (Mudah), 2 (Sedang), 3 (Sulit).PENTING: Karena kita menggunakan koma (,) sebagai pemisah kolom CSV, jangan gunakan koma (,) di dalam teks pertanyaan atau opsi Anda.Contoh Tampilan di Google Sheets / ExcelABCD1qoadifficulty2Ibukota Indonesia adalah?Jakarta\\Bandung\\SurabayaJakarta13Siapa penemu Teori Relativitas?Isaac Newton\\Albert EinsteinAlbert Einstein34Planet terdekat dari Matahari?Venus\\Merkurius\\BumiMerkurius2Metode 1: Menggunakan Google Sheets (Direkomendasikan)Langkah 1: Buat dan Isi SheetBuka sheets.google.com dan buat Sheet baru.Isi 4 header kolom: q, o, a, difficulty.Isi soal-soal Anda. Ingat, untuk kolom o, gunakan \\ sebagai pemisah (contoh: OpsiA\\OpsiB\\OpsiC).Langkah 2 (A): Menyimpan sebagai File Lokal (untuk Fase A-F)Klik File > Download > Nilai yang dipisahkan koma (.csv).Ganti nama file yang di-download menjadi FaseA.csv (atau FaseB.csv, dst.).Pindahkan file ini ke folder proyek Anda.Langkah 2 (B): Menerbitkan Link CSV (untuk Fitur Host/Kuis Kustom)Klik File > Bagikan > Publikasikan ke web.Pilih Sheet Anda dan ganti format menjadi Nilai yang dipisahkan koma (.csv).Klik Publikasikan dan salin link yang muncul.Tempelkan link ini di dalam game (Panel Host) untuk membuat kuis kustom.
